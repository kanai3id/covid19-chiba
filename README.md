## 開発者向け情報

### 環境構築の手順

- 必要となるNode.jsのバージョン: 14.16.0以上

**yarn を使う場合**
```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```
